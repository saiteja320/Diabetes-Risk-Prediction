# Diabetes Risk Prediction

Machine Learning project for predicting diabetes risk using the PIMA Indians Diabetes Dataset.

## Features
- Feature Engineering
- SMOTE
- StandardScaler
- PCA
- Random Forest Classifier

## Performance
- ROC-AUC Score: 0.83

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Imbalanced-learn
